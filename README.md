## Tecnologías Utilizadas

- HTML
- CSS (Tailwind CSS)
- JavaScript

## Instalación y Uso

1. Clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.

## Funcionalidades Clave

- Intersection observer para animaciones basadas en el scroll.
- Conceptos del patron MVC. 
- ESModule.
